# Generate parameters or some data for convolutional layers.

## Build.
```sh
cd build
cmake ../ -DCMAKE_CXX_COMPILER="/usr/local/gcc-9.3.0/bin/g++" -DCMAKE_C_COMPILER="/usr/local/gcc-9.3.0/bin/gcc"
make -j
```

## Run
```sh
a
```
